# Claude Plugins

A collection of plugins for Claude Cowork.

## Installation

```bash
claude plugins add ryankee/plugins
```

Or install a specific plugin:

```bash
claude plugins add ryankee/plugins/personal-os
```

## Available Plugins

| Plugin | Description |
|--------|-------------|
| [personal-os](./personal-os) | AI-powered task management with goal-driven prioritization. Capture thoughts in a backlog, process with smart deduplication, and align tasks with your goals. |
